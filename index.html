Welcome to the PowerPoint OS

An Operating System created using only Microsoft Office 365 PowerPoint (Not affiliated) which has full access to basic applications along with multiple themes and a great area to play around in without ruining your computer. Get stuck? Download a new copy!

Open source

Providing you accept to give credits, you can use this file to create a new version of PowerPoint OS, and it's free! You may send an email to me (email found in README.md) and submit your file to be displayed in the repository.
